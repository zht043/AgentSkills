# npu-basics

昇腾 NPU 基础命令 skill，提供设备状态监控、版本查询、设备指定、进程管理和日志排查能力。

## 使用方式

在 Claude Code 对话中直接描述需求即可，agent 会根据 SKILL.md 中的命令指引执行。

### Prompt 示例

```
查看当前服务器的 NPU 状态
```

```
查一下这台机器的 CANN 版本和驱动版本
```

```
我只想用第 0 和第 1 张卡，帮我设置环境变量
```

```
NPU 好像卡住了，帮我排查一下
```

```
查看哪些进程占用了 NPU 显存
```

## 注意事项

- 需要在已安装昇腾 NPU 驱动的 Linux 服务器上执行
- 远程场景需要配合 SSH 远程执行能力
- `npu-smi set -t reset` 会中断所有占用进程，使用前需确认
- CANN 版本查询路径因安装方式不同而异，skill 提供多种查找方法
